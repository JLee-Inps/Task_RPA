# Cursor 커맨드 사용법

## Git 자동 커밋

Cursor의 터미널이나 커맨드 팔레트에서 다음 명령어를 사용하세요:

### 기본 사용법

```bash
npm run git-commit "커밋 메시지"
```

또는

```bash
node scripts/git-auto-commit.js "커밋 메시지"
```

### 브랜치 지정

```bash
npm run git-commit "커밋 메시지" "브랜치명"
```

### Push 없이 커밋만

```bash
npm run git-commit "커밋 메시지" "" false
```

## 동작 방식

1. 로컬에서 Git commit 실행
2. Git push 실행 (옵션)
3. 커밋 정보를 백엔드 API로 전송
4. GPT로 커밋 내용 요약
5. 업무 목록에 자동 추가

## 환경 변수 설정

`.env` 파일에 다음을 추가하세요:

```env
API_URL=http://localhost:3001
JWT_TOKEN=your-jwt-token  # 선택사항 (로그인 후 토큰)
```

## Cursor 커맨드 팔레트에서 사용

1. `Cmd+Shift+P` (Mac) 또는 `Ctrl+Shift+P` (Windows/Linux)
2. "Terminal: Run Command" 선택
3. `npm run git-commit "메시지"` 입력
